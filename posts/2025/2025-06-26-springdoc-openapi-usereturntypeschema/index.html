@ApiResponse 어노테이션의 useReturnTypeSchema 속성#

  • 컨트롤러 메서드의 반환 타입을 자동으로 추론하여 OpenAPI 스펙의 response schema 로 사용
@Operation(summary = "사용자 조회", responses = {
    @ApiResponse(responseCode = "200", description = "조회 성공", useReturnTypeSchema = true)
})
